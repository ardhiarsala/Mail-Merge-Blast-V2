Developed as a modified version to the Google Apps Script Mail Merge Sample App by Martin Hawksey.

Current features include:

  • Beautiful email bodies based on Google Docs formatting
  • Mail subject, cc, bcc, and sender name customizations directly on the 'parameter' Sheet
  • Personalized attachments from Google Drive Document, Presentation or PDF files (using file IDs as variable)

To learn how to use this app and blast away your emails, refer to my Github
